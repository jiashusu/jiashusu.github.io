    先带你快速使用一遍 Git 管理你的项目文件

    发表于 2022-01-19 分类于 git

    现在你已经知道 Git 是分布式管理的工具，和集中式不同，在我们电脑就可以直接管理，速度飞快，因为我们根本不需要每次操作都连接到那台集中服务器上面去。

    和别的教程跟你讲一堆有的没的屁话理论不同，帅b先带你走一遍 Git 的流程，让你对 Git 先有一个大概的认知，以后再跟你说 Git 知识点的时候你才更加容易懂。请注意，从现在开始，你应该要动手了，不要光看，看一两遍是没什么卵用的，你要操作，自己动手才是王道。

    接下来，包括以后所有的教程，我都会事先运行一遍，觉得完全没有问题，才会写到教程中来，所以你放心食用，无毒无害，健康爽爽。

    下载安装Git

    你可以到以下 Git 官网下载 Git ：

    https://git-scm.com/downloads 

    Windows 下载安装方式是傻瓜式的，相信你懂，Linux 的安装方式官网也有介绍，认准你的 Linux 发行版本，按以下命令一两下就能搞定：

    Debian/Ubuntu 系统

    # apt-get install git

    Fedora 系统

    # yum install git

    下载安装完成之后，你打开你的命令行，输入 git – -version ，如果有显示版本号就说明你安装成功了。

    当然，市面上有很多 Git 客户端工具，就是有界面操作 Git 的那种，但是小帅b不允许你使用这些客户端的工具，你要使用命令行操作，只有你会了 Git 的命令使用，才能更加了解 Git 的整个操作流程。这样你以后遇到问题也更加容易排查问题，我们可是要做 20% 的人，怎么能用 GUI ？所以还是老老实实的使用命令行吧。
